チュートリアル
==================================

このチュートリアルでは、Himmelblau関数の最小化問題を例として ODAT-SE および ODAT-SE-template ライブラリの使い方を解説します。あわせて、ユーザが独自の関数を定義して解析を行う方法を説明します。

逆問題解析アルゴリズムはグリッド探索の ``mapper`` モジュールを使います。その他のアルゴリズムについては ODAT-SE のマニュアルを参照してください。
